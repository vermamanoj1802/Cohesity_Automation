# Agent_Register

This directory contains resources and scripts related to agent registration automation for Cohesity environments.

## Contents

- **win_register_API.yaml**  
  YAML file defining the API specification for registering Windows agents. This file can be used as a reference for API integrations or automation scripts.

## Usage

1. **Review the API Spec:**  
   Open `win_register_API.yaml` to understand the available endpoints, request parameters, and expected responses for Cohesity agent registration.

2. **Integrate or Automate:**  
   Use the API specification as a foundation for building automation scripts or integrating the registration workflow into your environment.

## Requirements

- Python (for automation scripts in this repository)
- Access to Cohesity API endpoints, as defined in the YAML file
